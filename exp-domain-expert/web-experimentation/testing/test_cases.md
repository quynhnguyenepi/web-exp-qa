# Test Cases for web-experimentation

## Overview
Test cases for web-experimentation domain expert skill.

## Test Categories

### 1. Knowledge Accuracy

| ID | Name | Input | Expected Outcome |
|----|------|-------|------------------|
| TC-001 | A/B test creation | "How do I create an A/B test?" | Accurate step-by-step instructions for experiment creation |
| TC-002 | Visual Editor changes | "How do I change font size in the Visual Editor?" | Correct VE workflow: select element, typography panel, set font size |
| TC-003 | Audience conditions | "What audience conditions are available?" | Lists browser, device, cookie, query param, custom JS, third-party integrations |
| TC-004 | Page activation modes | "What are the different page activation modes?" | Describes immediate, polling, callback, manual, and URL changed |
| TC-005 | Stats Engine | "How does Stats Engine determine significance?" | Explains sequential testing, always-valid p-values, false discovery rate control |

### 2. Edge Cases

| TC-007 | Feature boundary | "How do I use feature flags in Web?" | Clarifies this is a Feature Experimentation concept, suggests routing |

| ID | Name | Input | Expected Outcome |
