# Sample Query: opal-chat

## Question
"What can Opal do when I'm reviewing experiment results?"

## Answer
When reviewing experiment results, Opal provides several AI-assisted capabilities:

### Result Summarization
- Opal can generate a **plain-language summary** of your experiment results.
- It highlights which variations are winning, losing, or inconclusive.
- It explains statistical significance in accessible terms (e.g., "Variation B increased conversion rate by 3.2% with 95% confidence").

### Experiment Review
- Opal can review your experiment setup and flag potential issues:
  - Low sample size warnings
  - Metric configuration concerns
  - Audience overlap with other running experiments
- It provides recommendations for next steps (e.g., "Consider running for 2 more weeks to reach significance on secondary metrics").

### How to Access
1. Navigate to the Results page of any experiment.
2. Click the Opal icon (AI assistant) in the interface.
3. Ask Opal a question like "Summarize these results" or "What should I do next?"

### Important Notes
- Opal's interpretations are AI-generated and should be validated against the raw data.
- Opal does not replace statistical rigor; it supplements it with accessible explanations.
- Opal can reference historical experiment data within the same project for context.
